# Contributing

Thanks for your interest in Synapse.

This repository is public for learning, review, and reference. Before sending changes, please keep the following rules in mind:

- Do not submit real API keys, license keys, project data, generated videos, or installer outputs.
- Keep changes focused and small.
- Follow the existing Python and frontend style.
- Do not remove or bypass the license activation flow.
- For large feature changes, open an issue or discussion first.

Runtime use of the official software still requires a valid license key.
