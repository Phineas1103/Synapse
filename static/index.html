选题内容
选题一：职业众生相

以职业为切口，描绘不同时代众生相，折射职业背后的人性与社会——

古代职人：历史质感+职业细节，仵作 / 讼师 / 匠人 / 医师/古代官职等

当代冷门职业：殡葬师/速记员/文物修复师/非遗传承人/乡村医生/餐厅主理人等

高压决策行业：急诊 / 法医 / 心理咨询 / 危机谈判等生死与伦理决策的一线

成熟影视作品参考：《入殓师》《我不是药神》《人生大事》《捉妖记》《法医秦明》《寿司之神》《非自然死亡》
选题二：地域文化

县城叙事：聚焦三四线城市与县域青年，描摹日常现实困境与人生抉择，展现被忽视的地域生态

城市空间：以城市空间为载体，结合人物内心情绪叙事，打造贴近日常、温情治愈的贴身陪伴感

地域特色：立足川渝、东北、西北、西南苗寨、闽南侨乡、藏区牧场等特色地域场景，还原本土人文风貌与独有的在地生活气息

成熟影视作品参考：《路边野餐》《平原上的夏洛克》《隐入尘烟》《三和人才市场：中国日结1500日元的年轻人们》《爱情神话》《可可西里》《冈仁波齐》
选题三：悬疑推理

社会派悬疑：借由犯罪事件，探讨复杂的道德与社会议题

密室推理：封闭空间或极端设定下的推理博弈，注重逻辑严密性和叙事反转

纪实改编：基于真实案件或社会事件改编，保留纪实质感，增加文学性叙事

历史悬案：依托历史上的未解疑案创作，适配多元悬疑化剧情重构与内容创作

解谜互动：沉浸式剧情解谜设计，融入趣味线索探索，增强观众参与感与剧情互动体验

成熟影视作品参考：《白日焰火》《暴雪将至》《烈日灼心》 《看不见的客人》《彗星来的那一夜》《十二公民》、《大宋提刑官》《神探狄仁杰》
选题四：情感叙事

人与自然/宠物：以动物或自然为情感纽带，探讨孤独、陪伴与生命意义

遗憾美学：不追求大团圆，以遗憾、错过、告别为情感核心，追求余韵悠长

文艺感受：以日常生活的琐碎与诗意构建叙事节奏，传递文艺质感

成熟影视作品参考：《五个扑水的少年》《阳光灿烂的日子》《少年的你》《致青春》《忠犬八公的故事》《海街日记》
选题五：志怪灵异

志怪新编：取材《聊斋》、《山海经》、《搜神记》等古典志怪，以现代视角重新解读人妖/人鬼关系中的人性寓言

神话解构：对经典神话IP（济公/哪吒/孙悟空/白蛇等）进行反套路重构，注入现代价值观（注：不能颠覆原著及其中人物的核心精神，不可魔改）

西方诸神：北欧诸神如奥丁/托尔/洛基、希腊诸神如宙斯/雅典娜/阿波罗、罗马诸神如朱庇特/维纳斯等诸神故事

成熟影视作品参考：《灵魂摆渡》《画皮》《妖猫传》《捉妖记》、《哪吒之魔童降世》《雷神 3：诸神黄昏》
选题六：创新形态

单元剧：由主角串联主线，每集独立故事、共享世界观或主题，单集完成度高

情景喜剧：依托固定拍摄场景，以群像角色为核心，主打轻松幽默风格，笑点中暗含现实思考，观赏性与反复回看属性强

伪纪录片：采用纪实纪录片拍摄手法，演绎虚构 / 半虚构剧情，弱化表演感，提升画面真实感与观众沉浸体验

成熟影视作品参考：《武林外传》《炊事班的故事》《爱情公寓》、《世界奇妙物语》《不思异：录像》《黑镜》
选题七：文化传承

历史切片：聚焦某位历史人物最艰难或最辉煌的一段经历，切片具备强戏剧冲突or命运转折点

民俗仪式：围绕傩戏、庙会、祭祀、婚丧嫁娶等民间仪式，构建具有地域特色的叙事空间

古典文化：以古典文学、诗词、书法、绘画为线索，进行现代重构或跨时空叙事

传统技艺：以唢呐、昆曲、京剧、刺绣、制瓷、木雕等传统手艺为叙事主线，展现匠人精神与时代的困境

成熟影视作品参考：《康熙王朝》《万历首辅张居正》《百鸟朝凤》《霸王别姬》《闪光少女》《长安三万里》
选题八：游戏剧情

无限流：进入诡异空间，空间内暗藏不明规则、未知危险与隐藏线索

时空循环：主角反复经历同一天 / 同一事件，困于重复节点，在试错中寻找真相

解谜互动：剧情联动观众，每集结尾留一个谜题，观众需要在评论区 / 弹幕中参与推理

网络游戏衍生：依托热门网络游戏IP或真实网络游戏场景，打造衍生剧情，打破虚拟与现实的边界，聚焦游戏世界与现实生活的联结

成熟影视作品参考：《唐朝诡事录》《开端》《十日终焉》
选题九：社会纪实

时代痛感：聚焦不同代际成年人的真实心境，中年危机/奥德赛时期/退休生活失序等现实成长痛点

道德困境：搭建极致冲突情境与两难抉择设定，将人物推入进退两难的处境，深挖人性抉择与内心挣扎

社会实验：紧扣当下热门社会议题，改编现实热议事件创作故事，采用无绝对善恶的开放式叙事，引发大众深度思考
