# Security Policy

## Sensitive data

Do not commit real API keys, license keys, local project data, build outputs, or generated installers.

The following files and directories are intentionally ignored:

- `user_config.json`
- `license_cache.json`
- `projects/`
- `dist/`
- `build/`
- `installer_output/`
- `release/`
- `tools/`
